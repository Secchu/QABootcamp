Project
=======
A ASP.NET MVC Application that is a basic CRUD application for Employees. The application uses Dynamodb as the persistent store and you
will need to manually create the Dynamodb database on your own AWS account. It is possible to test the application locally using 
Dynamodb local by making a few modifications to project.
